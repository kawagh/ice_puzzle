# ice_puzzle
simple 2D-grid game made with [Ebiten](https://ebiten.org/)

![](./img.png)
## run
```
go run main.go
```
or
```
./ice_puzzle
```

## solver
- solver.py solve the puzzle from text file from below format
```
H W
sx sy
gx gy
s..
...
..g
```
